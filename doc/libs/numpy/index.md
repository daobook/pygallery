# NumPy

[NumPy] 是 Python 中科学计算的基础包。它是一个 Python 库，提供了多维数组对象、各种派生对象（如掩码数组和矩阵），以及一系列用于快速操作数组的例程，包括数学运算、逻辑运算、形状操作、排序、选择、输入输出、离散傅里叶变换、基本线性代数、基本统计操作、随机模拟等。
